Comprimi e converti automaticamente le immagini caricate nella libreria media di WordPress in JPG, PNG o WEBP.

== Description ==
FullCompression ti permette di selezionare il formato immagine desiderato (JPG, PNG, WEBP) e abilita la compressione automatica al momento del caricamento nella media library.

== Installation ==
1. Carica il plugin nella directory `/wp-content/plugins/`.
2. Attivalo tramite il menu 'Plugin' in WordPress.
3. Vai su **FullCompression** nella barra laterale e scegli le impostazioni desiderate.
